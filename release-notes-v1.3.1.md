# solar-layout-card v1.3.1

## Added

- **Playback speed controls.** The day-playback now has 1x, 2x and 4x buttons next to the
  play button, and the base speed is a bit quicker.
- **Footer bar in the visual editor.** The weather, forecast and total entities can now be
  set from the editor UI (previously YAML only), with a "Show total" checkbox. The footer
  bar appears automatically as soon as you fill in one of the entities.

## Updating

Update via HACS and then do a hard browser refresh (or clear the `.gz` in
`www/community/solar-layout-card/`) if you don't see the change immediately.
