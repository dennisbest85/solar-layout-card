# solar-layout-card v1.3.2

## Fixed

- The weather condition in the footer bar was always shown in English (e.g. "Sunny"). The
  weather conditions now follow the Home Assistant language (Dutch, German, English;
  otherwise English), like the rest of the card.

## Updating

Update via HACS and then do a hard browser refresh (or clear the `.gz` in
`www/community/solar-layout-card/`) if you don't see the change immediately.
