# Solar Layout Card v1.0.3

Nieuwe weergave-opties en een documentatie-fix.

## Nieuw
- **Configureerbare kleuren (Enphase-stijl)**: kleur voor "uit / geen zon" (standaard donkerblauw) en voor maximale opbrengst (standaard lichtblauw). De kleur loopt vloeiend tussen deze twee op basis van opbrengst t.o.v. de Wp van dat paneel. Tekstkleur past zich automatisch aan voor leesbaarheid.
- **Zoom**: verklein de opstelling van 100% tot 40% via een knop op de card. Standaard-zoom stel je in de editor in.

## Gewijzigd
- Realistischer paneelraster (6x10 cellen, 10x6 in landscape) i.p.v. 8 grove vakken.

## Opgelost
- README verscheen niet in HACS door een ongeldige byte in het bestand. README is opnieuw opgeslagen als schone UTF-8.
