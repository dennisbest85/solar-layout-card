# Solar Layout Card v1.0.0

Eerste release van de Solar Layout Card.

## Highlights
- **Legplan-weergave**: plaats je zonnepanelen zoals ze op het dak liggen.
- **Live opbrengst**: koppel per paneel een sensor; de waarde en eenheid worden in het paneel getoond.
- **Kleur op opbrengst**: panelen kleuren mee met de PV-opbrengst t.o.v. de ingestelde referentie (Wp/W).
- **Portrait & landscape**: elk paneel afzonderlijk te draaien.
- **Drag & drop editor**: sleep panelen op een raster, snap-to-grid, per paneel label en sensor kiezen.

## Installatie
1. Voeg de repo toe in HACS als custom repository (categorie: Dashboard).
2. Installeer en herlaad je browser.
3. Voeg de card toe aan je dashboard en gebruik de visuele editor.

## Bekende aandachtspunten
- Referentiewaarde bepaalt de kleurschaal; stel deze in op het piekvermogen (Wp) of de verwachte piek in W.
